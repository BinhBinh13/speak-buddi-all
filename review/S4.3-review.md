## Findings

### Minor (không block PASS)

1. **`handleRetry` không reset attemptId**: Sau khi retry, nếu user nộp lại sẽ gặp 409 (attempt đã submitted). Code đã có comment `// TODO S4.5` — acceptable vì S4.5 sẽ handle lọc attempts.

2. **`grammar_mapping` luôn `is_correct = True` ở BE**: Đây là quyết định có chủ ý (self-grading, ghi chú trong code). S4.4 có thể bổ sung proper pair-check nếu cần.

3. **`unansweredIds` là `Set` truyền qua props**: React không track thay đổi bên trong `Set` khi so sánh reference. Implementation dùng `new Set(...)` để tạo Set mới mỗi lần validate → re-render đúng. ✅
