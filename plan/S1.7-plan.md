## 1. Tổng quan

### Luồng end-to-end

```
[Login page]
  └── "Quên mật khẩu?" link → /forgot-password

[ForgotPasswordPage]
  └── Nhập email → POST /api/auth/forgot-password
        ├── Email tồn tại → tạo token, lưu in-RAM, gửi email qua Gmail SMTP
        └── Email không tồn tại → vẫn trả 200 (chống user enumeration)

[Email của user]
  └── Click link "{FRONTEND_URL}/reset-password?token=<token>"

[ResetPasswordPage]  ← đọc ?token= từ URL
  └── Nhập password mới + xác nhận → POST /api/auth/reset-password
        ├── Token hợp lệ + pass valid → cập nhật hash, xóa token → redirect /login
        └── Token sai/hết hạn → hiển thị error + link quay lại /forgot-password
```

**Token:** `secrets.token_urlsafe(32)` — 256-bit ngẫu nhiên, TTL 30 phút, one-time use.  
**Security:** Không log raw token; đồng nhất message cho email tồn tại/không tồn tại (chống user enumeration).  
**Email:** Gmail SMTP qua `smtplib` (built-in Python, không cần cài thêm package).  
**Backend hiện tại:** vẫn dùng MOCK_USERS in-RAM (nhất quán với S1.4–S1.6); migration sang DB thật ở S1.8.

#### 2.3.5 Helper: gửi email qua Gmail SMTP

```python
def _send_reset_email(to_email: str, token: str) -> None:
    """
    Gửi email đặt lại mật khẩu qua Gmail SMTP (smtplib built-in).
    Nếu SMTP_USER/SMTP_PASS chưa set → chỉ log WARNING, không raise
    (cho phép dev/test không cần cấu hình email thật).
    """
    if not SMTP_USER or not SMTP_PASS:
        log.warning("RESET_EMAIL skip: SMTP chưa cấu hình — set SMTP_USER và SMTP_PASS trong .env")
        return

    reset_link = f"{FRONTEND_URL}/reset-password?token={token}"

    msg = MIMEMultipart("alternative")
    msg["Subject"] = "Đặt lại mật khẩu SpeakBuddi"
    msg["From"]    = SMTP_USER
    msg["To"]      = to_email

    html_body = f"""
<div style="font-family:'Be Vietnam Pro',Arial,sans-serif;max-width:520px;margin:auto;padding:32px;background:#fcf8ff;border-radius:12px;">
  <h2 style="color:#3525cd;margin-bottom:8px;">SpeakBuddi</h2>
  <h3 style="color:#1b1b24;">Đặt lại mật khẩu</h3>
  <p style="color:#464555;">Chúng tôi nhận được yêu cầu đặt lại mật khẩu cho tài khoản của bạn.</p>
  <p style="color:#464555;">Link này có hiệu lực trong <strong>30 phút</strong>.</p>
  <a href="{reset_link}"
     style="display:inline-block;margin:16px 0;padding:12px 28px;background:#3525cd;color:#fff;border-radius:12px;text-decoration:none;font-weight:600;">
    Đặt lại mật khẩu
  </a>
  <p style="color:#777587;font-size:13px;">
    Nếu bạn không yêu cầu đặt lại mật khẩu, hãy bỏ qua email này.
  </p>
  <hr style="border-color:#c7c4d8;margin-top:24px;"/>
  <p style="color:#777587;font-size:12px;">© 2024 SpeakBuddi. Nâng tầm tiếng Anh của người Việt.</p>
</div>
"""
    msg.attach(MIMEText(html_body, "html", "utf-8"))

    try:
        with smtplib.SMTP(SMTP_HOST, SMTP_PORT) as server:
            server.ehlo()
            server.starttls()
            server.login(SMTP_USER, SMTP_PASS)
            server.sendmail(SMTP_USER, to_email, msg.as_string())
        log.info("RESET_EMAIL sent via Gmail SMTP (không log email người nhận)")
    except smtplib.SMTPAuthenticationError:
        log.error("RESET_EMAIL auth failed: kiểm tra SMTP_USER và App Password")
    except Exception as exc:
        log.error("RESET_EMAIL error: %s", type(exc).__name__)
    # Không raise — FE vẫn nhận 200; user không biết có lỗi gửi mail
```

#### 2.3.6 Endpoint: `POST /api/auth/forgot-password`

```python
@app.post("/api/auth/forgot-password")
async def forgot_password(req: ForgotPasswordRequest):
    email = req.email.strip().lower() if req.email else ""

    if not email or not _validate_email(email):
        raise HTTPException(status_code=400, detail="Email không hợp lệ.")

    # Tìm user — nếu không tồn tại vẫn trả 200 (chống user enumeration)
    user = _find_user_by_email(email)
    if user:
        # Hủy token cũ của user (nếu có) để tránh nhiều token còn hiệu lực
        for t, data in list(PASSWORD_RESET_TOKENS.items()):
            if data["email"] == email:
                del PASSWORD_RESET_TOKENS[t]

        token = secrets.token_urlsafe(32)
        PASSWORD_RESET_TOKENS[token] = {
            "email": email,
            "exp":   time.time() + RESET_TOKEN_TTL,
        }
        log.info("RESET_TOKEN created for user (không log email/token)")
        _send_reset_email(email, token)

    return {"message": "Nếu email tồn tại trong hệ thống, chúng tôi đã gửi hướng dẫn đặt lại mật khẩu."}
```

#### 2.3.7 Endpoint: `POST /api/auth/reset-password`

```python
@app.post("/api/auth/reset-password")
async def reset_password(req: ResetPasswordRequest):
    token  = req.token.strip() if req.token else ""
    new_pw = req.new_password or ""

    # Validate token
    record = PASSWORD_RESET_TOKENS.get(token)
    if not record:
        raise HTTPException(status_code=400, detail="Yêu cầu đặt lại mật khẩu không hợp lệ hoặc đã được sử dụng.")
    if record["exp"] < time.time():
        del PASSWORD_RESET_TOKENS[token]
        raise HTTPException(status_code=400, detail="Yêu cầu đặt lại mật khẩu đã hết hạn. Vui lòng gửi lại yêu cầu.")

    # Validate password mới
    if not _validate_password(new_pw):
        raise HTTPException(
            status_code=400,
            detail="Mật khẩu phải có ít nhất 8 ký tự và 1 chữ số.",
        )

    # Cập nhật password_hash trong MOCK_USERS
    email = record["email"]
    user  = _find_user_by_email(email)
    if not user:
        del PASSWORD_RESET_TOKENS[token]
        raise HTTPException(status_code=400, detail="Tài khoản không tồn tại.")

    user["password_hash"] = hashlib.sha256(new_pw.encode()).hexdigest()

    # Xóa token (one-time use)
    del PASSWORD_RESET_TOKENS[token]
    log.info("RESET_PASSWORD ok user=%s", user["id"])  # không log email/password/token

    return {"message": "Mật khẩu đã được cập nhật thành công."}
```

### 2.4 `speak-buddi-be/db/schema_core.sql` — thêm bảng (cho S1.8 migration)

Thêm sau bảng `user_session`:

```sql
-- ─── 5. password_reset_token ─────────────────────────────────────────────────
-- Lưu token hash (§4.5: không lưu raw token) — dùng khi S1.8 migrate sang DB
CREATE TABLE IF NOT EXISTS password_reset_token (
    id         UUID         PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id    UUID         NOT NULL REFERENCES users (id) ON DELETE CASCADE,
    token_hash VARCHAR(64)  NOT NULL UNIQUE,  -- SHA-256 hex của raw token
    expires_at TIMESTAMPTZ  NOT NULL,
    used       BOOLEAN      NOT NULL DEFAULT FALSE,
    created_at TIMESTAMPTZ  NOT NULL DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_prt_user_id ON password_reset_token (user_id);
CREATE INDEX IF NOT EXISTS idx_prt_expires ON password_reset_token (expires_at)
    WHERE NOT used;
```

| Rủi ro | Mức độ | Giảm thiểu |
