## 1. Mục tiêu & phạm vi

Xây dựng màn **Luyện phát âm** với khả năng **ghi âm qua microphone bằng MediaRecorder API** và **xử lý đầy đủ các lỗi mic** (denied / not found / unsupported browser). Story này chỉ lo phần thu âm + UI + error handling; chưa gọi API chấm điểm (S6.2) và chưa lưu audio (S6.3).

"Done" = người dùng vào màn luyện phát âm, thấy từ cần đọc + phiên âm + nút Ghi âm; bấm ghi âm thì state máy chạy qua `idle → recording → processing → done`; khi mic bị từ chối/không có/trình duyệt không hỗ trợ thì hiển thị đúng thông điệp tiếng Việt trong SRS §5.2; sau khi dừng ghi âm hiển thị placeholder "Đang chấm điểm..." (chỗ chờ S6.2 cắm vào).

**Ngoài phạm vi (không làm ở S6.1):** gọi API chấm phát âm + render score/feedback/syllable (S6.2), lưu audio/metadata vào DB/storage (S6.3), backend endpoint chấm điểm. Phần "Score Display" trong mockup thuộc S6.2 — S6.1 chỉ dựng khung và placeholder.

#### State machine ghi âm (`useMediaRecorder.js`)

Enum trạng thái:
```
idle        → chưa ghi, sẵn sàng
recording   → đang thu (MediaRecorder.state === "recording")
processing  → đã stop, đang gom Blob (ngắn) → chuẩn bị bàn giao cho S6.2
done        → có audioBlob hợp lệ; hiển thị ScorePlaceholder ("Đang chấm điểm...")
error       → kèm errorCode (enum) + message §5.2
```

Chuyển trạng thái:
```
idle      --press record-->  (gọi getUserMedia)
   ├─ thành công           --> recording
   └─ DOMException         --> error (denied | notfound | unsupported)
recording --press stop / max-duration / silence--> processing
processing--blob ok------->  done   (chừa hook bàn giao Blob cho S6.2)
processing--blob rỗng/ngắn-> error (noaudio)
error/done--press retry--->  idle
```

API hook trả về:
```js
const {
  status,        // "idle"|"recording"|"processing"|"done"|"error"
  error,         // { code, message } | null
  elapsedMs,     // số ms đã ghi (cho RecordingTimer)
  audioBlob,     // Blob | null — dành cho S6.2
  start, stop, reset,
} = useMediaRecorder({ maxDurationMs: 15000 });
```
Chi tiết kỹ thuật:
- Kiểm tra hỗ trợ trước: nếu `!navigator.mediaDevices?.getUserMedia` **hoặc** `typeof MediaRecorder === "undefined"` → `error: unsupported` (không gọi getUserMedia).
- `getUserMedia({ audio: true })`, lưu `MediaStream` để dọn dẹp.
- `new MediaRecorder(stream)`, gom `dataavailable` vào mảng chunks, ghép thành `Blob` khi `stop`.
- Tự dừng khi đạt `maxDurationMs` (gắn quota free của SRS sẽ ở story khác; ở đây chỉ giới hạn 1 lần ghi để tránh ghi vô hạn).
- **Cleanup bắt buộc:** `stream.getTracks().forEach(t => t.stop())` khi stop/unmount để tắt đèn mic và giải phóng thiết bị.
- Không log/giữ audio ra ngoài (SRS §4.5 — không log audio).

#### Map lỗi mic (`micErrors.js`)
```js
export const MIC_ERROR = { DENIED, NOTFOUND, UNSUPPORTED, NOAUDIO };
export function mapGetUserMediaError(err) {
  // err.name: NotAllowedError/SecurityError → DENIED
  //           NotFoundError/OverconstrainedError/DevicesNotFoundError → NOTFOUND
  //           NotSupportedError / không có API → UNSUPPORTED
  // → trả { code, message } với message lấy đúng chuỗi §5.2
}
```
Bảng ánh xạ DOMException → enum:
| DOMException.name | enum | message (§5.2) |
|---|---|---|
| `NotAllowedError`, `SecurityError`, `PermissionDeniedError` | DENIED | `🎤 Bạn chưa cấp quyền microphone. Vào Settings → trình duyệt → cho phép mic.` |
| `NotFoundError`, `DevicesNotFoundError`, `OverconstrainedError` | NOTFOUND | `🎤 Không tìm thấy microphone. Kiểm tra thiết bị.` |
| `NotSupportedError` / thiếu API | UNSUPPORTED | `🎤 Trình duyệt không hỗ trợ. Vui lòng dùng Chrome hoặc Edge.` |
| (blob rỗng/quá ngắn sau khi ghi) | NOAUDIO | `🎤 Không nghe thấy gì. Hãy thử nói lại.` |

## 5. Test & cách verify

**Thủ công (Chrome/Edge DevTools):**
- **AC-08-01 (happy path):** Cho phép mic → bấm ghi → state `recording`, timer chạy, có pulse ring → bấm dừng → `processing` → `done` → hiển thị "Đang chấm điểm..."; xác nhận `audioBlob` có `size > 0` (log tạm trong dev, KHÔNG commit log).
- **AC-08-02 denied:** Trong site settings của trình duyệt set Microphone = Block (hoặc bấm "Block" ở prompt) → bấm ghi → state `error`, hiện đúng message denied.
- **AC-08-02 notfound:** DevTools → tắt/ngắt thiết bị mic (hoặc dùng `navigator.mediaDevices.getUserMedia` stub ném `NotFoundError`) → hiện message missing.
- **Unsupported:** Tạm override `window.MediaRecorder = undefined` (hoặc test trên trình duyệt không hỗ trợ) → vào trang/bấm ghi → hiện message unsupported, KHÔNG gọi getUserMedia.
- **No audio:** Ghi rồi dừng ngay (blob rỗng/quá ngắn) → hiện message "Không nghe thấy gì".
- **Retry:** Ở mọi state error/done bấm "Thử lại" → quay về `idle`, mic track đã được tắt (đèn mic tắt).
- **Cleanup:** Rời trang khi đang ghi → mic tracks dừng (không còn icon mic đang dùng trên tab).

**UI/responsive:** so sánh với `screen.png`; kiểm 375/768/1024/1440px; nút mic ≥ 44px; màu/font đúng `DESIGN.md`.
