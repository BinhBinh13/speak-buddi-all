## 3. Thiết kế kỹ thuật

### 3.1 Backend (`speak-buddi-be/`)

**Endpoint mới:** `POST /api/translate`

```
Request:  { "text": "Hello world" }
Response: { "translation": "Xin chào thế giới" }
Errors:
  400  { "detail": "Input is empty" }       — text rỗng/whitespace
  401  Unauthorized                          — không có JWT
  502  { "detail": "Translation service error" } — Anthropic lỗi
```

**Files cần tạo:**

| File | Nội dung |
|------|----------|
| `schemas/translate.py` | `TranslateRequest(text: str)`, `TranslateResponse(translation: str)` |
| `services/translate_service.py` | `translate_text(text: str) -> str` — gọi Anthropic với system prompt dịch |
| `routers/translate.py` | `POST /api/translate` với `Depends(current_user)` |

**Sửa:**
- `main.py`: thêm `from routers import translate` + `app.include_router(translate.router)`

**System prompt cho dịch thuật:**
```
You are a professional English-to-Vietnamese translator.
Translate the following English text to Vietnamese naturally and accurately.
Return ONLY the Vietnamese translation — no explanation, no extra text, no quotes.
```

**Model:** `claude-haiku-4-5-20251001`, `max_tokens=500` (đủ cho đoạn văn trung bình).

**Auth:** `Depends(current_user)` — cùng pattern với `learning.py`, `quiz.py`.

**Convention:**
- Router: `prefix="/api"`, `tags=["translate"]`
- Logger: `log = logging.getLogger("speakbuddi.translate")`
- Log: `log.info("TRANSLATE  len=%d", len(text))` — không log nội dung text.

### 3.2 Frontend (`speak-buddi/`)

**Cấu trúc feature:**
```
speak-buddi/src/features/translate/
├── TranslatePage.jsx          — page container (wrap AppLayout)
├── components/
│   └── TranslatorCard.jsx     — card dịch thuật chính
└── services/
    └── translateService.js    — gọi POST /api/translate
```

**TranslatePage.jsx:**
- Wrap `<AppLayout>`
- Tiêu đề: "Dịch thuật" (tiếng Việt, theo quy ước UI tiếng Việt)
- Sub-text: "Dịch văn bản tiếng Anh sang tiếng Việt nhanh chóng."
- Render `<TranslatorCard />`

**TranslatorCard.jsx** (bám mockup `dich_thuat_desktop/code.html`):
- **Language bar:** "English" | swap icon (disabled, chỉ EN→VI) | "Tiếng Việt"
- **Hai panel cạnh nhau (md: flex-row, mobile: flex-col):**
  - Panel trái — Input:
    - `<textarea>` placeholder "Nhập văn bản tiếng Anh..."
    - Char counter `n / 5000`, disable submit nếu > 5000
    - Nút clear (×) reset input
    - Nút "Dịch" ở dưới (primary button)
    - Disable nút + border đỏ + warning text khi input rỗng (AC-07-02)
  - Panel phải — Output:
    - Placeholder text "Bản dịch sẽ hiện ở đây" khi chưa có kết quả
    - Loading skeleton khi đang gọi API
    - Hiển thị bản dịch khi thành công
    - Nút Copy (copy bản dịch vào clipboard)
    - Error state khi API thất bại
- **State:** `inputText`, `translation`, `loading`, `error`, `touched`

**validateInput():**
```js
if (!inputText.trim()) {
  return "Vui lòng nhập văn bản cần dịch.";
}
if (inputText.length > 5000) {
  return "Văn bản không được vượt quá 5000 ký tự.";
}
return null;
```

**translateService.js:**
```js
// POST /api/translate
export async function translateText(text) {
  const res = await apiClient.post("/api/translate", { text });
  return res.data.translation;
}
```
Dùng `apiClient` từ `speak-buddi/src/shared/api/client.js` (axios instance có JWT interceptor).

**App.jsx — thêm route:**
```jsx
import TranslatePage from "../features/translate/TranslatePage";
// Trong ProtectedRoute:
<Route path="/translate" element={<TranslatePage />} />
```

**DashSidebar.jsx — thêm nav item:**
```jsx
import { LuLanguages } from "react-icons/lu";
// Thêm vào NAV_ITEMS sau "Quiz":
{ label: "Translate", Icon: LuLanguages, path: "/translate" },
```

**AppLayout.jsx — thêm PAGE_TITLES:**
```jsx
"/translate": { title: "Dịch thuật", icon: "" },
```

**AppLayout** và **MobileBottomNav** cần thêm Translate vào bottom nav nếu phù hợp (xem MobileBottomNav hiện có mấy item).
