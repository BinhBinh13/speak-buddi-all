## Tổng quan

S5.2 thêm tính năng lưu và xem lịch sử dịch. Mỗi lần dịch thành công (với user đã đăng nhập), kết quả được tự động lưu vào bảng `translation_history` (PostgreSQL). Người dùng có thể xem 20 mục gần nhất ngay trên trang Dịch thuật. Guest không lưu và không thấy phần lịch sử.

Cấu trúc công việc chia làm 3 lớp:

1. **Backend – DB + Repository:** Tạo bảng, viết repository.
2. **Backend – Router + Schema:** Endpoint save (hook vào POST /api/translate) và endpoint GET history.
3. **Frontend – UI section lịch sử:** Hiển thị danh sách trong `TranslatePage`, gọi API khi mount.

## Chi tiết implement

### Backend

#### 1. Schema SQL — bảng `translation_history`

Chạy migration thêm bảng. Không có ORM model file riêng — dùng raw SQL nhất quán với pattern hiện tại.

```sql
-- Chạy trực tiếp trên DB hoặc ghi vào db/migrations/add_translation_history.sql
CREATE TABLE IF NOT EXISTS translation_history (
    id          UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id     UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    source_text TEXT NOT NULL,
    target_text TEXT NOT NULL,
    created_at  TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX IF NOT EXISTS idx_translation_history_user_created
    ON translation_history (user_id, created_at DESC);
```

**Lý do chọn:**
- `UUID` PK nhất quán với toàn bộ DB (xem `users`, `quiz_attempt`…).
- `ON DELETE CASCADE` – xóa user thì history tự xóa.
- Index trên `(user_id, created_at DESC)` tối ưu query top-20.
- Không có `updated_at` vì history là immutable.

#### 2. `speak-buddi-be/repositories/translate_repo.py` (TẠO MỚI)

```python
# speak-buddi-be/repositories/translate_repo.py
# Repository cho translation_history (S5.2)
# Pattern: bám sát repositories/quiz_repo.py

from sqlalchemy import text
from sqlalchemy.ext.asyncio import AsyncSession


async def save_history(
    db: AsyncSession,
    user_id: str,
    source_text: str,
    target_text: str,
) -> None:
    """Lưu 1 bản dịch vào translation_history. Không trả về gì — fire-and-forget."""
    await db.execute(
        text("""
            INSERT INTO translation_history (user_id, source_text, target_text)
            VALUES (CAST(:uid AS UUID), :src, :tgt)
        """),
        {"uid": user_id, "src": source_text, "tgt": target_text},
    )


async def get_history(
    db: AsyncSession,
    user_id: str,
    limit: int = 20,
) -> list[dict]:
    """Lấy top {limit} bản dịch gần nhất của user. Mới nhất trước."""
    r = await db.execute(
        text("""
            SELECT id::text,
                   user_id::text,
                   source_text,
                   target_text,
                   created_at
            FROM   translation_history
            WHERE  user_id = CAST(:uid AS UUID)
            ORDER  BY created_at DESC
            LIMIT  :lim
        """),
        {"uid": user_id, "lim": limit},
    )
    return [dict(row) for row in r.mappings().all()]
```

#### 4. `speak-buddi-be/routers/translate.py` (SỬA)

Hai thay đổi:

**a) Inject DB vào `POST /api/translate` và gọi `save_history` sau khi dịch thành công:**

```python
import logging

from fastapi import APIRouter, Depends, HTTPException
from sqlalchemy.ext.asyncio import AsyncSession

from auth.deps import current_user
from db.connection import get_db
from repositories import translate_repo
from schemas.translate import (
    TranslateRequest,
    TranslateResponse,
    TranslationHistoryItem,
    TranslationHistoryResponse,
)
from services.translate_service import translate_text

log = logging.getLogger("speakbuddi.translate")

router = APIRouter(prefix="/api", tags=["translate"])


@router.post("/translate", response_model=TranslateResponse)
async def translate(
    body: TranslateRequest,
    user: dict = Depends(current_user),
    db: AsyncSession = Depends(get_db),
) -> TranslateResponse:
    if not body.text.strip():
        raise HTTPException(status_code=400, detail="Input is empty")

    try:
        result = translate_text(body.text)
    except Exception as exc:
        log.error("TRANSLATE_ERROR  %s", type(exc).__name__)
        raise HTTPException(status_code=502, detail="Translation service error") from exc

    # Lưu history — lỗi DB không được fail kết quả dịch (AC-07-03 edge case)
    user_id: str | None = user.get("sub")
    if user_id:
        try:
            await translate_repo.save_history(
                db,
                user_id=user_id,
                source_text=body.text,
                target_text=result,
            )
        except Exception as exc:
            log.error("HISTORY_SAVE_ERROR  %s", type(exc).__name__)
            # Không raise — tiếp tục trả kết quả dịch bình thường

    return TranslateResponse(translation=result)
```

**Lưu ý quan trọng:** `translate_text()` hiện là hàm **đồng bộ** (không có `async def`). Khi router chuyển sang `async def translate`, phải bọc `translate_text` bằng `asyncio.to_thread()` hoặc giữ endpoint là synchronous `def` nhưng dùng `run_in_threadpool`. Cách đơn giản nhất: **đổi router thành `async def` và wrap `translate_text` bằng `asyncio.to_thread`**:

```python
import asyncio

result = await asyncio.to_thread(translate_text, body.text)
```

**b) Thêm endpoint `GET /api/translate/history`:**

```python
@router.get("/translate/history", response_model=TranslationHistoryResponse)
async def get_history(
    user: dict = Depends(current_user),
    db: AsyncSession = Depends(get_db),
) -> TranslationHistoryResponse:
    """
    Trả 20 bản dịch gần nhất của user đang login (AC-07-03).
    401 nếu không có JWT (Depends(current_user) tự xử lý).
    """
    user_id: str | None = user.get("sub")
    if not user_id:
        raise HTTPException(status_code=401, detail="Invalid token payload")

    items_raw = await translate_repo.get_history(db, user_id=user_id, limit=20)
    items = [TranslationHistoryItem(**row) for row in items_raw]
    return TranslationHistoryResponse(items=items)
```

```jsx
// speak-buddi/src/features/translate/components/TranslationHistory.jsx
import { useEffect, useState } from "react";
import { COLORS, FONTS } from "../../../shared/constants/theme";
import { getTranslationHistory } from "../services/translateService";

function formatDate(isoString) {
  if (!isoString) return "";
  const d = new Date(isoString);
  return d.toLocaleString("vi-VN", {
    day: "2-digit", month: "2-digit",
    hour: "2-digit", minute: "2-digit",
  });
}

export default function TranslationHistory({ refreshTrigger }) {
  const [items, setItems]     = useState([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    let cancelled = false;
    setLoading(true);
    getTranslationHistory()
      .then((data) => { if (!cancelled) { setItems(data); setLoading(false); } })
      .catch(() => { if (!cancelled) setLoading(false); }); // ẩn lỗi DB
    return () => { cancelled = true; };
  }, [refreshTrigger]);

  // ... render theo mockup (section header + grid cards)
}
```

## Edge cases & rủi ro

| Case | Xử lý |
|---|---|
| Lỗi DB khi `save_history` | `try/except` trong router, log error, **không raise** — kết quả dịch vẫn trả về bình thường |
| User là guest (không có JWT) | Route `/translate` đã nằm trong `ProtectedRoute` → không thể vào; nhưng nếu token hết hạn giữa chừng → apiClient tự redirect `/login`; `GET /api/translate/history` trả 401 |
| `user.get("sub")` trả None | Check `if user_id:` trước khi gọi `save_history` — bỏ qua save nếu payload bất thường |
| `asyncio.to_thread` wrap `translate_text` | `translate_text` dùng `client.messages.create` (blocking). Trong `async def` endpoint phải wrap để không block event loop |
| DB chưa có bảng `translation_history` | Migration SQL phải chạy trước khi deploy. Ghi rõ trong implement log. |
| `source_text` rất dài (5000 ký tự) | PostgreSQL `TEXT` không giới hạn — OK. FE đã có `MAX_CHARS = 5000`. |
| History load chậm | Fetch chạy song song với render trang (`useEffect` mount), không block UI dịch. Loading state hiện skeleton. |
| `TranslationHistoryItem.created_at` timezone | SQLAlchemy trả `TIMESTAMPTZ` về Python `datetime` có timezone. Pydantic serialize thành ISO string. FE dùng `new Date(isoString)` → xử lý đúng. |
