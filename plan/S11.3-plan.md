## 1. Mục tiêu & phạm vi

Cho Admin **xuất báo cáo** dạng **Excel (.xlsx)** hoặc **PDF** theo loại dữ liệu và bộ lọc thời gian (và gói thanh toán với báo cáo doanh thu), **tải file về trình duyệt**, và **ghi lịch sử export** vào DB để xem lại trên màn Reports.

"Done" khi: Admin vào `/admin/reports` → chọn loại báo cáo + khoảng ngày (+ gói nếu doanh thu) + định dạng xlsx/pdf → bấm **Xuất báo cáo** → nhận file tải xuống hợp lệ; bảng **Lịch sử export** hiển thị bản ghi mới (thời gian, loại, định dạng, trạng thái, admin).

**Trong phạm vi S11.3:** migration `report_export_history`, API export + list history, engine sinh xlsx/pdf đồng bộ (< 30s), trang FE `/admin/reports`, route sidebar Reports.
**Ngoài phạm vi:** export bất đồng bộ/queue, lưu file lên S3 (`file_url` persistent), CSV/JSON (mockup có nhưng SRS chỉ yêu cầu Excel/PDF), email gửi file, export toàn bộ raw DB.

**Migration — `db/migrations/007_report_export_history.sql`:**

Port từ `speak-buddi-docs/db/schema-draft.sql` sang PostgreSQL (UUID admin, TIMESTAMPTZ):

```sql
CREATE TABLE IF NOT EXISTS report_export_history (
    id              BIGSERIAL PRIMARY KEY,
    admin_user_id   UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    report_type     TEXT NOT NULL CHECK (report_type IN ('revenue','users','learning','ai_usage')),
    export_format   TEXT NOT NULL CHECK (export_format IN ('xlsx','pdf')),
    filter_params   JSONB,
    file_name       VARCHAR(255),          -- tên file client tải (không bắt buộc URL storage)
    status          TEXT NOT NULL DEFAULT 'pending'
                    CHECK (status IN ('pending','completed','failed')),
    error_message   VARCHAR(500),
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    completed_at    TIMESTAMPTZ
);
CREATE INDEX IF NOT EXISTS idx_report_export_admin_created
    ON report_export_history (admin_user_id, created_at DESC);
```

> `file_url` trong SRS draft → MVP dùng **stream trực tiếp**; cột `file_name` thay cho audit. `file_url` có thể thêm sau nếu lưu object storage.

**Schema Pydantic — `schemas/analytics.py` (thêm):**

```python
ReportTypeName = Literal["revenue", "users", "learning", "ai_usage"]
ExportFormatName = Literal["xlsx", "pdf"]

class ExportRequestIn(BaseModel):
    report_type: ReportTypeName
    export_format: ExportFormatName
    from_date: date = Field(..., alias="from")
    to_date: date = Field(..., alias="to")
    plan_id: UUID | None = None          # chỉ revenue
    granularity: GranularityName = "day" # revenue: bucket trong sheet tóm tắt

class ExportHistoryItemOut(BaseModel):
    id: int
    report_type: ReportTypeName
    export_format: ExportFormatName
    filter_params: dict | None
    file_name: str | None
    status: Literal["pending", "completed", "failed"]
    error_message: str | None
    created_at: datetime
    completed_at: datetime | None

class ExportHistoryListOut(BaseModel):
    items: list[ExportHistoryItemOut]
    total: int
```

**Repo — `repositories/report_export_repo.py` (MỚI):**

- `create_export(db, admin_id, report_type, export_format, filter_params) -> int` — insert `pending`
- `mark_completed(db, export_id, file_name)` / `mark_failed(db, export_id, error_message)`
- `list_exports(db, admin_id, *, limit=20, offset=0) -> tuple[list, total]` — admin chỉ xem lịch sử của mình (hoặc toàn hệ thống nếu PO muốn — **MVP: mọi admin xem tất cả**, filter optional `admin_id` sau)

**Service — `services/report_export_service.py` (MỚI):**

Orchestrate: validate → query data → build bytes → update history.

| `report_type` | Nguồn dữ liệu (tái dùng) | Nội dung file |
|---|---|---|
| `revenue` | `analytics_repo.get_revenue_filtered` + query chi tiết `payment_transaction` | Sheet1: tóm tắt (tổng ₫, số GD, plan, range); Sheet2/detail: mã GD, ngày, gói, amount |

**Helpers — `services/export_builders.py` (MỚI, tùy chọn gộp vào service):**

- `build_xlsx(report_type, payload) -> bytes`
- `build_pdf(report_type, payload) -> bytes` — header SpeakBuddi, bảng đơn giản reportlab `Table`

**Router — `routers/admin_analytics.py` (thêm):**

```
POST /api/admin/analytics/export
  Body: ExportRequestIn
  Response: StreamingResponse (application/vnd.openxmlformats-officedocument.spreadsheetml.sheet
            hoặc application/pdf)
  Headers: Content-Disposition: attachment; filename="speakbuddi-{type}-{from}-{to}.xlsx"
  Flow: create pending → generate → mark completed/failed → stream bytes

GET /api/admin/analytics/exports?limit=20&offset=0
  Response: ExportHistoryListOut
```

Validation POST (mirror S11.2):

- `from_date` / `to_date` bắt buộc; `from > to` → 400
- `plan_id` chỉ hợp lệ khi `report_type=revenue`; plan không tồn tại → 404
- `report_type=ai_usage` vẫn export được (placeholder content)

**Đăng ký:** không cần router mới — đã mount `admin_analytics` trong `main.py`.

**UX flow:**

1. Mount `/admin/reports` → load history (20 mới nhất) + plans (cho revenue).
2. Admin chọn loại + from/to + format → **Xuất báo cáo** → loading trên nút.
3. FE `fetch` POST với `response.blob()` → tạo object URL → trigger download → toast thành công.
4. Refresh history table.
5. Lỗi 400/404 → hiển thị `detail` tiếng Việt inline (pattern `adminValidation`).

**Download pattern (`reportExportService.js`):**

```javascript
export async function exportReport(payload) {
  const res = await apiClient("/api/admin/analytics/export", {
    method: "POST",
    body: JSON.stringify(payload),
    // apiClient cần trả raw Response hoặc blob — mở rộng option raw: true nếu chưa có
  });
  // parse Content-Disposition filename; return { blob, fileName }
}
```

> Nếu `apiClient` hiện chỉ parse JSON → thêm nhánh `responseType: 'blob'` (chỉ cho export) — ghi rõ trong implement log.
