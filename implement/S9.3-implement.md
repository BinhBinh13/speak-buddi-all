## Cách test thủ công

1. Chạy schema: `psql … -f db/schema_crawler.sql`
2. Env: `LANGEEK_CRAWL_ENABLED=true`, `LANGEEK_USE_FIXTURE=true` (mặc định)
3. Login admin → `/admin/crawler` → Bắt đầu crawler → toast thành công
4. `GET /api/admin/crawler/sync-status` — `words_upserted_this_week` > 0
5. Query DB: `SELECT * FROM topic_word WHERE source='langeek' AND is_active=true`
6. Student GET `/api/topics/{id}/words` — thấy từ mới (badge "Mới" nếu chưa progress)
7. Live crawl: `LANGEEK_USE_FIXTURE=false`, `npm run crawl` trong speak-buddi-scrape (cần Playwright + DOM selectors)
