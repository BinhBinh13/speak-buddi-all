## Edge cases da xu ly

| Edge case | Xu ly |
|---|---|
| id_token het han / sai signature | `google-auth` raise `ValueError` → HTTP 400 "id_token khong hop le hoac het han." |
| Loi network khi fetch JWKS | `Exception` khac `ValueError` → HTTP 502 "Loi xac thuc voi Google." (khong log token) |
| GOOGLE_CLIENT_ID chua set | `RuntimeError` ngay khi verify — fail ro rang |
| email khong co trong id_token | Fallback: `f"{google_sub}@google.oauth"` — du unique |
| name khong co trong id_token | Fallback: phan truoc @ cua email |
| Tai khoan suspended | HTTP 403 "Tai khoan da bi khoa." |
| GSI chua load (window.google undefined) | FE reject: "Google Identity Services chua tai xong. Vui long thu lai." |
| VITE_GOOGLE_CLIENT_ID chua set | FE reject: "Cau hinh Google OAuth chua duoc thiet lap." |
| One Tap bi chan (browser/privacy settings) | Fallback: renderButton ket hop click tu dong |
| Nguoi dung huy popup | `cancel_on_tap_outside: true`; neu One Tap khong hien thi → fallback button |
| Google button chua render kip | FE reject: "Dang nhap Google khong kha dung. Vui long thu lai." |
| OAuth user (password_hash = None) dung /api/auth/login | Fail tu nhien — `compare_digest(sha256_hash, None)` nem loi, tra 401 |
