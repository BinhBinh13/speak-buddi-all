# Implement — S11.3: Export Excel/PDF + lưu lịch sử

- **Story:** S11.3
- **Epic:** EPIC-11 — Admin: Phân tích & Báo cáo

## Tóm tắt

Trang `/admin/reports` cho Admin xuất báo cáo **Excel/PDF** theo loại dữ liệu + khoảng ngày (+ gói với doanh thu), tải file qua stream, và xem **lịch sử export** trong DB. BE ghi `report_export_history` sau mỗi lần xuất thành công/thất bại.

## Quyết định implement

- **Định dạng:** Chỉ `xlsx` và `pdf` theo SRS (mockup có CSV/JSON — bỏ qua).
- **Download:** `reportExportService.js` dùng `fetch` + `blob` riêng (không sửa `apiClient` JSON).
- **Lưu file:** Stream trực tiếp — không lưu disk/S3; cột `file_name` trong history để audit.
- **Lịch sử:** MVP mọi admin xem toàn bộ export (GET `/exports`).
- **AI usage:** Placeholder giống dashboard S11.1 khi `is_available=false`.
- **Zero state:** File vẫn tạo với dòng "Chưa có dữ liệu…".
