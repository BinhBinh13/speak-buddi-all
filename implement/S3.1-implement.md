## Canh bao truoc khi khoi dong app

1. **Cai dependency:** `cd speak-buddi-be && pip install -r requirements.txt` (asyncpg + sqlalchemy da uncomment)
2. **Chay schema theo thu tu:**
   ```bash
   psql -U postgres -d speakbuddi -f db/schema_core.sql
   psql -U postgres -d speakbuddi -f db/schema_learning.sql
   ```
   (Ca hai idempotent — chay lai nhieu lan an toan)
3. **Set .env:** dam bao `DATABASE_URL=postgresql+asyncpg://postgres:postgres@localhost:5432/speakbuddi` trong `speak-buddi-be/.env`
4. **Luu y schema_learning.sql vs schema_core.sql:** schema_core.sql hien tai da chua cac bang learning tu phien implement truoc. schema_learning.sql se la no-op doi voi cac bang do (IF NOT EXISTS bao ve), nhung la nguon DDL chinh thuc cho nhom Learning content.

## Cach test thu cong (sau khi chay schema)

```sql
-- 1. Kiem tra 5 bang ton tai
\dt level topic tag topic_word topic_word_tag

-- 2. Kiem tra 6 level A1-C2
SELECT code, name, display_order FROM level ORDER BY display_order;
-- Ket qua mong doi: 6 rows A1..C2

-- 3. Kiem tra soft delete (topic van ton tai sau khi set is_active=false)
UPDATE topic SET is_active = false WHERE slug = 'daily-routines';
SELECT count(*) FROM topic;                          -- phai la 2 (row van con)
SELECT count(*) FROM topic WHERE is_active = true;  -- phai la 1
UPDATE topic SET is_active = true WHERE slug = 'daily-routines';  -- restore

-- 4. Loc tu vung theo level (FK + denormalize)
SELECT w.word, w.phonetic, w.meaning_vi
FROM topic_word w
JOIN topic t ON w.topic_id = t.id
WHERE t.level_id = (SELECT id FROM level WHERE code = 'A1')
  AND w.is_active = true;
-- Ket qua mong doi: 5 tu (hello, goodbye, name, meet, introduce)

-- 5. Kiem tra chong trung (UNIQUE topic_id, word)
INSERT INTO topic_word (topic_id, level_id, word, meaning_vi)
VALUES (
    (SELECT id FROM topic WHERE slug = 'greetings-introductions'),
    (SELECT id FROM level WHERE code = 'A1'),
    'hello', 'trung lap'
);
-- Ket qua mong doi: ERROR unique constraint violated

-- 6. Kiem tra trigger updated_at
UPDATE topic SET description = 'test trigger' WHERE slug = 'greetings-introductions';
SELECT description, updated_at FROM topic WHERE slug = 'greetings-introductions';
-- updated_at phai doi sang thoi diem hien tai
```
