1. Khởi động BE: `cd speak-buddi-be && uvicorn main:app --reload --port 8000`
2. Khởi động FE: `cd speak-buddi && npm run dev`
3. Đăng nhập (user có token hợp lệ) → vào `/translate`
4. **Test AC-07-01:** Nhập "Hello world" → bấm "Dịch" → output hiện "Xin chào thế giới" (hoặc tương đương)

7. **Test auth guard:** Truy cập `/translate` khi chưa đăng nhập → redirect về `/login`.
8. **Test responsive:** Resize browser về 375px → 2 panel xếp dọc; nút "Dịch" vẫn tương tác được.
