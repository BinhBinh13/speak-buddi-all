## Đối chiếu Acceptance Criteria

- [x] **AC-08-01** — Khi quyền mic được cấp: `start()` gọi `getUserMedia`, tạo `MediaRecorder`, gom Blob; khi `blob.size ≥ 1000` chuyển state `done` + lưu `audioBlob` sẵn cho S6.2. `ScorePlaceholder` hiển thị "Đang chấm điểm...".
- [x] **AC-08-02** — Mic bị từ chối hoặc không tìm thấy: `mapGetUserMediaError()` map DOMException → `{ code, message }`, hiển thị qua `MicErrorAlert`.
- [x] **Mic denied** → `"🎤 Bạn chưa cấp quyền microphone. Vào Settings → trình duyệt → cho phép mic."` — khớp SRS §5.2 (NotAllowedError/SecurityError/PermissionDeniedError).
- [x] **Mic missing** → `"🎤 Không tìm thấy microphone. Kiểm tra thiết bị."` — khớp SRS §5.2 (NotFoundError/DevicesNotFoundError/OverconstrainedError).
- [x] **Unsupported browser** → `"🎤 Trình duyệt không hỗ trợ. Vui lòng dùng Chrome hoặc Edge."` — check `!navigator.mediaDevices?.getUserMedia || typeof MediaRecorder === "undefined"` trước khi gọi API.
- [x] **No audio** → `"🎤 Không nghe thấy gì. Hãy thử nói lại."` — khi `blob.size < 1000` sau khi dừng ghi.

## Cách test thủ công

1. Chạy `cd speak-buddi && npm run dev`, vào `http://localhost:5173` (đăng nhập trước).
2. Điều hướng đến `/pronunciation`.
3. **Happy path (AC-08-01):** Cho phép mic khi trình duyệt hỏi → bấm nút mic → pulse ring xuất hiện, timer chạy → bấm dừng (hoặc đợi 15s) → "Đang chấm điểm..." hiển thị.
4. **Denied (AC-08-02):** Vào Site Settings của Chrome → chặn mic → reload → bấm nút → banner đỏ "Bạn chưa cấp quyền microphone..." hiện.
5. **Notfound:** Ngắt thiết bị mic → bấm nút → banner "Không tìm thấy microphone...".
6. **Unsupported:** Trong DevTools console: `window.MediaRecorder = undefined` → bấm nút → banner "Trình duyệt không hỗ trợ...".
7. **No audio:** Cấp quyền mic, bấm ghi ngay rồi dừng ngay (< 1 giây) → banner "Không nghe thấy gì...".
8. **Retry:** Ở bất kỳ state `error` hoặc `done` → bấm "Thử lại" → quay về `idle`, nút mic trở lại màu primary; kiểm tra tab trình duyệt không còn icon mic.
9. **Cleanup khi rời trang:** Đang ghi → navigate sang `/dashboard` → icon mic trên tab tắt.
10. **Responsive:** Kiểm tra 375px (mobile), 768px, 1024px, 1440px — layout center, nút ≥ 44px.
