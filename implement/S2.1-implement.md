## Cách test thủ công

1. Start BE: `cd speak-buddi-be && uvicorn main:app --reload --port 8000`
2. Start FE: `cd speak-buddi && npm run dev`
3. Đăng ký user mới → tự động redirect `/onboarding`.
4. Bước 1: chọn level (ví dụ B1) → nút Tiếp tục enable. Không chọn → disabled.
5. Bước 2: topic load từ `GET /api/onboarding/topics?level=B1`. Chọn ít nhất 1 → Tiếp tục enable.
6. Bước 3: chọn phút/ngày.
7. Bước 4: chọn số từ → nút "Hoàn tất & Bắt đầu học" → gọi `POST /api/onboarding` → redirect `/dashboard`.
8. Vào lại `/onboarding` sau khi done → guard redirect `/dashboard`.
9. `GET /api/auth/me` → kiểm tra `onboarding_completed: true`, `level: "B1"`, `daily_minutes`, `words_per_session`.
10. `SELECT target_level, interests, daily_minutes, words_per_session FROM user_profile WHERE user_id = '...'` → có dữ liệu đúng.
11. Test BR09: `curl -X POST /api/onboarding -H "Authorization: Bearer ..." -d '{"level":"X9","topics":[],"daily_minutes":10,"words_per_session":10}'` → 400 + message đúng.
