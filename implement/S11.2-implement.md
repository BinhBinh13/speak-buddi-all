## Quyết định implement

- **Nguồn doanh thu:** `payment_transaction.status='success'`, `amount_vnd`, `paid_at` — `is_estimated=false`.
- **Preset thời gian:** Tính trên DB (`NOW()`) qua `resolve_revenue_date_range` — tránh lệch TZ FE.
- **Custom range:** Cần cả `from` và `to`; validate `from ≤ to` → 400 tiếng Việt.
- **Chart revenue:** Dùng `/revenue` (points từ filter); chart users giữ `/timeseries` + range 7d/30d/year.
- **Overview KPI:** `get_revenue_stats` cũng dùng transaction thật (đồng bộ S11.2).
