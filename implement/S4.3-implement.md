## Cách test thủ công

1. Khởi động BE: `cd speak-buddi-be && uvicorn main:app --reload --port 8000`
2. Khởi động FE: `cd speak-buddi && npm run dev`
3. Đăng nhập, vào một bài kiểm tra có câu hỏi (đường dẫn `/quiz/<testId>`)
4. Làm bài nhưng **bỏ qua một số câu** (để `handleSkip` hoặc skip page) → bấm "Nộp bài" ở màn Review
5. Kiểm tra: toast đỏ "Vui lòng trả lời tất cả câu hỏi trước khi nộp" xuất hiện, câu chưa trả lời bị highlight đỏ
6. Bấm vào câu highlight đỏ → quay lại câu đó để trả lời
7. Trả lời hết tất cả câu → bấm "Nộp bài" → navigate đến `/quiz/:testId/result/:attemptId`
8. Kiểm tra DB: bảng `quiz_attempt` có row mới status=`submitted`, bảng `quiz_attempt_answer` có row từng câu
