## Quyết định kỹ thuật quan trọng

1. **GrammarMappingQuestion shuffle** — `Math.random` không được phép trong `useMemo` (lint rule `react-hooks/purity`). Giải pháp: hàm `shuffleItems()` khai báo ngoài component (pure helper), gọi qua `useState(() => shuffleItems(parsedItems))` lazy init — chỉ chạy 1 lần khi mount.

2. **3D Flip CSS** — Dùng `perspective: 1000px` trên container, `transformStyle: preserve-3d` + `rotateY(180deg)` trên inner + `backfaceVisibility: hidden` trên cả 2 mặt. Fallback tự nhiên: trình duyệt không hỗ trợ → flip vẫn xảy ra (ẩn mặt lật) nhờ CSS.

3. **GrammarMapping format** — Answers có format `"term → definition"` (split bằng ` → `). Đây là convention của `generate_questions_from_topic_words()` trong service.

4. **QuizRunner key prop** — Component được reset hoàn toàn khi chuyển câu nhờ `key={currentQuestion?.id ?? currentIndex}`. Đảm bảo state cục bộ (isFlipped, selectedId, inputValue, pairs) được reset sạch.

5. **VocabularyPage quiz button** — Dùng `alert()` tạm thời khi không có test (thay vì toast system chưa xây). TODO (S4.x): thay bằng toast component chuẩn.

6. **is_correct trong response** — BE trả về `is_correct` trong `QuizQuestionOut`. Acceptable cho MVP học tập; S4.3 sẽ validate server-side.

7. **Màn kết quả tạm** — Hiển thị số câu trả lời + flashcard known count. S4.4 sẽ thay bằng trang kết quả chi tiết có chấm điểm.

## Cách test thủ công

1. Start BE: `cd speak-buddi-be && uvicorn main:app --reload --port 8000`
2. Kiểm tra Swagger UI `http://localhost:8000/docs` → thấy 3 endpoint `/api/tests*`
3. `GET /api/tests?topic_id=<uuid>` → 200 list tests (cần topic có test trong DB)
4. `GET /api/tests/<test_id>` → 200 metadata test; test không active → 404
5. `GET /api/tests/<test_id>/questions` → 200 list questions + answers
6. Start FE: `cd speak-buddi && npm run dev`
7. Đăng nhập → Học từ vựng → Chọn topic có từ vựng → Nút "Làm bài kiểm tra" active
8. Click nút → navigate `/quiz/<testId>` (nếu có test) hoặc alert "chưa có"
9. Trong `/quiz/<testId>`: xem lần lượt flashcard, MC, fill_blank, grammar_mapping
10. Flashcard: click thẻ → lật 3D → nút "Cần ôn lại" / "Đã biết"
11. Multiple choice: click option → border-primary → nút Tiếp theo active
12. Fill blank: gõ / click chip word bank → nút Kiểm tra active
13. Grammar mapping: click trái → click phải → badge màu → Kiểm tra
14. Sau câu cuối → màn kết quả tạm với nút "Làm lại" và "Quay lại học từ vựng"
