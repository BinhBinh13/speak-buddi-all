## Quyết định kỹ thuật: `word_count`

Bảng `topic` trong schema KHÔNG có cột `word_count` riêng. Đã dùng subquery `COUNT(*)` trên `topic_word WHERE topic_id = t.id AND is_active = TRUE` — trả số từ active thực tế theo thời điểm request. Không hard-code 0.

## Cách test thủ công

1. Khởi động BE: `cd speak-buddi-be && uvicorn main:app --reload --port 8000`
2. Đăng nhập để lấy JWT, sau đó hoàn thành onboarding (level A1, chọn topic "greetings-introductions") qua `POST /api/onboarding`.
3. Gọi `GET /api/roadmap` với header `Authorization: Bearer <token>` (hoặc dùng Swagger UI `/docs`).
   - Kỳ vọng: 200, `level="A1"`, `nodes` có 1 node `slug="greetings-introductions"`, `is_interest=true`, `order_index=0`, `word_count=5`.
4. Test empty state: tạo user mới chưa onboard → `GET /api/roadmap` → 200, `nodes=[]`, `total_topics=0`, `level=""`.
5. Test 401: gọi không có header JWT → 401 Unauthorized.
6. Test inactive topic: `UPDATE topic SET is_active=FALSE WHERE slug='greetings-introductions'` → `GET /api/roadmap` → `nodes=[]`.
7. Test interests-first: onboarding level A2 (nếu có nhiều topic), chọn 1 topic → gọi roadmap → topic đã chọn xuất hiện đầu với `is_interest=true`.
