## Cách test thủ công

1. Đăng nhập `admin@speakbuddi.com` / mật khẩu mock (S1.8)
2. Vào `/admin/topics` → tạo/sửa topic; thử trùng tên → lỗi inline ở field name
3. Vào `/admin/vocabulary` → thêm/sửa từ; thử trùng từ trong topic → lỗi inline
4. Vào `/admin/tests` → Create New Test → editor → thêm câu MC, Save
5. Thử MC không chọn đáp án đúng → badge Incomplete + không lưu
6. Đăng nhập student → vào `/admin/topics` → redirect `/`
7. `curl -H "Authorization: Bearer <student_token>" http://localhost:8000/api/admin/topics` → 403
